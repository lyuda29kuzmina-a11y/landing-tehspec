Кейсы

Примеры работ

Задача Вся аналитика в проекте с оборотом 20+ млн руб./мес. считалась вручную. Нужна система KPI по каждому спикеру, отделу и сегменту оплат.
Решение Сегментировала заказы по предложениям (конверсия каждого спикера), тегам (горячие, дожим, апсейл, партнёрка…) и причинам отмен. Настроила дашборды с ключевыми показателями по каждому отделу и сотруднику.
Результат Полностью автоматическая аналитика вместо ручного учёта. Проект начал точечно дорабатывать воронку продаж и маркетинговую стратегию на основе реальных цифр.
Задача Запустить автовебинарную воронку одновременно на Bizon 365 и GetCourse.
Решение Провела живой вебинар через OBS + VK, обработала запись, собрала сценарий в Bizon (комментарии, ссылки, слайды), адаптировала для GetCourse (виджеты, команды). Настроила расписание, аналитику присутствия и автоматическую передачу данных участников из Bizon → GetCourse.
Результат Автовебинар запускается по расписанию без участия команды. Лиды, кликнувшие по кнопке и не оставившие заявку, автоматически попадают в нужные сегменты GetCourse.
Задача Автоматически выдавать сертификаты по завершении обучения и управлять доступами и рассылками по разным условиям прохождения курса.
Решение Настроила шаблоны дипломов и процессы выдачи сертификата после выполнения всех уроков. Добавила процессы: выдача доступов к курсам, отправка сообщений на ключевых этапах, передача данных в Senler для сегментации по статусу оплаты.
Результат Сертификаты, доступы и письма выдаются автоматически — команда освобождена от ручной работы. Senler получает корректные сегменты для разных цепочек воронок.
Задача Собрать обратную связь от студентов для построения персонализированной воронки продаж следующего продукта.
Решение Создала многошаговую анкету в GetCourse (1 вопрос = 1 шаг), ограничила доступ к следующему уроку до полного заполнения, вывела все ответы в Google Sheets в структурированном виде.
Результат 100% студентов заполнили анкету до перехода к следующему модулю. Данные в Google Sheets стали основой для сегментации и персонализации дальнейшей воронки.
